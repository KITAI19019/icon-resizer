# 🛠️ICO 缩放调节器

这是一个轻量级、完全在浏览器本地运行的图标处理工具。专门用于解决 Windows 任务栏或桌面图标大小不一、视觉不统一的问题（例如 GitHub 图标总是比 Chrome 小一圈）。

### 🌟 核心功能
- **一键上传**：支持 PNG、JPG、WebP 格式。
- **实时缩放**：通过滑块自由调整图标主体大小，去除多余留白。
- **视觉参考**：内置“标准参考圆”，助你对齐 Chrome、Gmail 等主流图标的视觉面积。
- **原生导出**：直接生成标准的 256x256 `.ico` 格式，支持透明度。
- **隐私安全**：所有图片处理均在本地浏览器 Canvas 完成，**不会上传到任何服务器**。

### 🛠️ 如何使用
1. **上传**：将你想要修改的图标文件拖入上传区。
2. **缩放**：拖动滑块，观察图标主体，使其边缘接近虚线“参考圆”。
3. **下载**：点击“保存并下载 ICO”，然后在 Windows 中替换原有的快捷方式图标即可。
